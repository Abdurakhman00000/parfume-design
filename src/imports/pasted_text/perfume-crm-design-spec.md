Ты — Senior Product Designer + UX/UI Architect + Art Director с опытом разработки e-commerce платформ и CRM систем для бизнеса.

Твоя задача:
спроектировать ПОЛНОЦЕННЫЙ профессиональный дизайн сайта + CRM системы для парфюмерного бизнеса в Кыргызстане.

ВАЖНО:
Это НЕ просто интернет-магазин.
Главная цель проекта — решить проблему потери клиентов при большом потоке заявок после рекламы и автоматизировать обработку заказов менеджерами.

────────────────────
КОНТЕКСТ БИЗНЕСА
────────────────────

Сейчас бизнес сталкивается с такими проблемами:

1. После запуска рекламы приходит очень много клиентов.
Например:
из 3000 написавших клиентов менеджеры успевают обработать только около 600.

2. Остальные клиенты теряются из-за хаоса в обработке заявок.

3. Бывают случаи мошенничества:
клиенты отправляют фейковые чеки об оплате.

Наша задача:
создать систему, которая:
— автоматизирует оформление заказа,
— автоматизирует оплату,
— собирает клиентов в CRM,
— помогает менеджерам быстро обрабатывать заявки,
— уменьшает потери клиентов,
— делает процесс максимально удобным и понятным.

────────────────────
ОСНОВНАЯ ЛОГИКА ПРОЕКТА
────────────────────

Будет сайт с:
— товарами,
— карточками товаров,
— категориями,
— фильтрацией,
— поиском,
— акциями,
— современным UX.

В карточке товара:
кнопка «Купить».

После нажатия открывается форма заявки:

— Имя Фамилия
— Номер телефона
— Адрес доставки
— Выбранный товар
— Способ оплаты

Для оплаты:
интеграция платежной системы Finik через QR-код.

После успешной оплаты:
клиент автоматически добавляется в CRM систему.

Пример записи:
— Имя клиента
— Номер
— Какой парфюм купил
— Адрес доставки
— Сумма заказа
— Статус заявки

────────────────────
ЛОГИКА CRM / СТАТУСЫ
────────────────────

Статусы клиента:

1. Новый
— клиент только оплатил заказ.

2. В обработке
— менеджер принял заявку.

3. Не дозвонились
— менеджер не смог связаться.

4. Не отвечает
— клиент не берет трубку.

5. Доставка отправлена

6. Успешно закрыт
— заказ успешно завершен.

Менеджер:
— принимает заявки,
— звонит клиенту,
— подтверждает адрес,
— обновляет статусы,
— контролирует доставку.
